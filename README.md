# Weather App

### This README is written using AI

This project is a simple desktop weather application built with Python and PyQt5. It lets users enter a city name and view the current temperature, weather description, and a matching emoji.

## Features
- Enter any city name
- Fetch live weather data from the OpenWeatherMap API
- Display current temperature in Celsius
- Show weather description and emoji
- Handle errors such as invalid city names, bad API responses, or connection issues

## Requirements
Install the following Python packages:

```bash
pip install requests PyQt5
```

## How to Run
1. Make sure Python is installed on your system.
2. Install the required packages.
3. Run the script:

```bash
python Weather.py
```

## Notes
- The app uses an OpenWeatherMap API key in the script.
- If the API key stops working, replace it with your own valid key.
- Make sure you have an active internet connection while using the app.
